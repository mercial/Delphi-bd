# Delphi-bd
Управление БД MS SQL через приложение Delphi.
Добавить новую запись:
<br>1)Файл -> Новый
<br>2)Кнопка "Добавить запись" на главной форме
<br>3)Кнока на компоненте BDNavigator
<br>4)TAB на DBGrid
<br><br>
Удалить запись:
<br>1)Кнопка "Удалить запись" на главной форме
<br>2)Кнока на компоненте BDNavigator
<br><br>
Изменить запись можно через компонент DBGrid На главной форме.
