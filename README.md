# Delphi-bd
Управление БД MS SQL через приложение Delphi.
Добавить новую запись:
1)Файл -> Новый
2)Кнопка "Добавить запись" на главной форме
3)Кнока на компоненте BDNavigator
4)TAB на DBGrid

Удалить запись:
1)Кнопка "Удалить запись" на главной форме
2)Кнока на компоненте BDNavigator

Изменить запись можно через компонент DBGrid На главной форме.
